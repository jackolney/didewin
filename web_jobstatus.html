Job status

Usage

web_jobstatus(x, cluster = valid_clusters()[[1]], state = "*", n = Inf)

Arguments

x
Either a username or configuration object
cluster
The cluster to check (default taken from configuration if passed as x).
state
State of jobs to check; default is all states ("*"). Other valid options are "Running", "Finished", "Queued", "Failed" and "Cancelled".
n
number of rows to return (default is Inf, which returns all available jobs).

Description

Get job status from the cluster
